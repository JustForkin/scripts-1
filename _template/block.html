This page lists characters in the following Unicode block and provides information about them.

This is not authoritative, peer-reviewed information – these are just notes I have gathered and copied from various places.

For a summary of the script and its use in writing systems, see the page Template script summary. For similar information related to other scripts, see the Script comparison table.

If you click on any red example text, you will see at the bottom right of the page a list of the characters that make up the example.

To find a character by codepoint, type #char0000 at the end of the URL in the address bar, where 0000 is a four-figure, hex codepoint number, all in uppercase. Or type the hex number here:

To view this page as intended, you need a Template font. This page comes with a Noto Sans Template webfont. Click the blue vertical bar at the bottom right of the page to apply other fonts, if you have them on your system. For transcriptions I recommend the excellent and free Doulos SIL font. Consonants

Basic consonants

ߒ

U+07D2 NKO LETTER N

See in UniView

Kangbe nasal syllable n̩

This is considered to be neither a consonant nor a vowel. It represents a syllabic alveolar or velar nasal. It doesn't take the nasal diacritic  ߲ [U+07F2 NKO COMBINING NASALIZATION MARK​].1

Basic tone marks

Kangbe tone mark

Tone mark indicating a short high tone.

Also used above consonants to represent sounds from other languages, particularly French and Arabic, as follows:

ߛ߫ ᵴ ߛ [U+07DB NKO LETTER SA] Arabic ص
ߜ߫ ɣ ߜ [U+07DC NKO LETTER GBA] Arabic غ
ߞ߫ q ߞ [U+07DE NKO LETTER KA] Arabic ق

Punctuation

߲

U+07F2 NKO COMBINING NASALIZATION MARK

See in UniView

Description in the Unicode standard:

→ (combining dot below - 0323)

Kangbe nasalisation mark

Applied to a vowel to indicate nasalisation, eg. ߣߌ߲߬.

Also used to indicate an ordinal number (except for the first), eg. ߂߲ second. When there are multiple digits in a number, the diacritic appears only under the last in sequence, eg. ߁߂߃߲ 123rd.1

For the first ordinal number, see  ߭ [U+07ED NKO COMBINING SHORT RISING TONE​].

Refs: [1] [Unicode] p738
