Tai Tham character notes

Updated Sun 21 Oct 2014 • tags bopomofo, chinese, scriptnotes

In this page I am exploring typing order when creating orthographic syllables in Tai Tham. It starts from work by Wyn Owen (p146), and builds on it.

The aim is to see whether there is an optimal order for input of characters, and to explore some anomalies. I start by mapping out what seems to work with a couple of fonts.

The starting point is marked with a coloured background. The main (onset) character in the syllable is indicated with a larger letter. (They don't always coincide.) V = vowel; C = consonant; M = medial consonant; + = sakot. The superscript numbers indicate the spoken order.
